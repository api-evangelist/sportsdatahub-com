# SportsDataHub.comThis is a repo for managing machine readable artifacts for the [SportsDataHub.com APIs](http://sportsdatallc.com/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [SportsDataHub.com APIs](http://sportsdatallc.com/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://sportsdatallc.com/)- [Developer](http://sportsdatallc.com/)- [Twitter](https://twitter.com/sportsdatahub)- [GitHub](https://github.com/SportsDataHub)The [APIs.json](https://github.com/api-evangelist/sportsdatahub-com/blob/master/apis.json) index provides links to all the relevant links for the SportsDataHub.com APIs, including the OpenAPI and Postman if they are present.
